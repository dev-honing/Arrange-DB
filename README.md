# Arrange-DB
클래스룸 자료를 예제로 DB에 데이터 저장하고 연습하기

## [#1. MariaDB 초기세팅](https://github.com/dev-honing/Arrange-DB/issues/1)
### npm 초기화
```
npm init -y
```

### MariaDB 패키지 설치
```
npm install mysql2
```

### .gitignore 세팅
node_modules/ 추가

## [#2. 서버 제작과 index.html 서빙](https://github.com/dev-honing/Arrange-DB/issues/2)
### Express 패키지 설치
```
npm install express
```
### app.js
Express로 서버를 제작
### index.html
테스트 데이터를 입력하고, 서빙 확인
